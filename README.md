# google-genaiの学習

Google Generative AI (google-genai) SDKの学習用リポジトリです。

## 使用技術

- **[uv](https://docs.astral.sh/uv/)**: Pythonパッケージマネージャー
- **[Ruff](https://docs.astral.sh/ruff/)**: Pythonのlinter/formatter
- **[Ty](https://docs.astral.sh/ty/)**: Python型チェッカー
- **[google-genai](https://googleapis.github.io/python-genai/)**: Google Generative AI SDK for Python

## セットアップ

```bash
cd app
uv sync
```

## 実行

```bash
cd app
uv run main.py
```
